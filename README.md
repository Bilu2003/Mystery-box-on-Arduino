# Mystery-box-on-Arduino
A mystery box project built with Arduino, featuring a button LED game, as well as self-opening boxes powered by electric motors.

The project was built using LEDs, buttons, a motor driver, and electric motors.

It involved designing and assembling the complete electronic system, including all connections and components. The software was written in Arduino (C++) and incorporates recursion to handle parts of the game logic.
